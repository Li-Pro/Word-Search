Dependencies
---
Requirements are stated in '\requirements.txt'.
(Use 'pip3 install -r requirements.txt' to install.)

Build
---
The main library is contained in 'diclib\'.
(Copy this folder when using.)

Run the 'demo.py' for demo.

Using
---
1. Copy the 'diclib' folder to your project.
2. In Python: 'import diclib'.
3. Refer to 'diclib\' for more details
